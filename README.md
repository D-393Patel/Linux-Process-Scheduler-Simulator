# Linux Process Scheduler Simulator (C)

A simple Operating Systems project that simulates common CPU scheduling algorithms used in Linux-like systems.

## Algorithms Implemented

- First Come First Serve (FCFS)
- Shortest Job First (SJF)
- Round Robin (RR)

## Concepts Demonstrated

- Process scheduling
- Waiting time & turnaround time
- CPU burst simulation
- Basic OS process management

## Compile

gcc scheduler.c -o scheduler

## Run

./scheduler
